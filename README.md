# Llanquihue Tour App

Gestión de tours turísticos mediante lectura de archivo .txt y ArrayList en Java.

## Descripción
Proyecto que demuestra la lectura de datos desde un archivo de texto, la creación de objetos y el uso de colecciones (ArrayList) con operaciones de recorrido y filtrado.

## Instrucciones para ejecutar
1. Abrir el proyecto en Apache NetBeans.
2. Ejecutar la clase `llanquihuetourapp.ui.Main`.
